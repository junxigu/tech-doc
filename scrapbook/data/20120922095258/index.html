简介

蚂蚁在线本次给大家带来一款强大的工具。谷歌提供的GoAgent是 一个使用Python和Google Appengine SDK编写的代理软件，配合chrome浏览器进行配置，很容易就可以高速浏览国外的一些学术网站。


工具/原料

  • Google App Engine

  • Chrome浏览器

  • goagent 1.7.10

步骤/方法

  1. 到Google App Engine ，如果还没有帐号，点击注册来申请一个帐号。

    注册Google App Engine 的帐号，演示中我未实质注册，因此后面可能存在一些需要手机验证的页面，具体请参考这里



  4. 点击 Create Application 来创建新的应用。


  6. 输入一个应用名，点击 Check Availability 检测可用后，点击 Create Application 来创建应用。

    应用创建成功。

  7. 到 goagent 主页下载其最新版本。

  8. 下载文件解压后，使用文本编辑器打开 goagent/local/proxy.ini，修改第8行的 appid = 为刚申请的应用名。

  9. 运行 goagent/server/uploader.bat 文件，根据提示输入 appid 以及 Gmail 的邮箱、密码等来进行上传。
    注意如果 Gmail 设置了二次密码验证，需要再去 Google 帐户管理中设置应用程序专用密码。

  10. 上传完成的话，Google App Engine 里面的应用列表中，Current Version 将不再是 None Deployed 状态。

  11. 运行 goagent/local/goagent.exe 程序。

  12. 运行 Chrome，安装[url]Proxy SwitchySharp[/url]插件。

  13. 在 SwitchySharp 选项中，选择导入导出分页卡，导入此设置 http://goagent.googlecode.com/files/SwitchyOptions.bak。

  14. 在 SwitchSharp 选项中，选择切换规则选项卡，点击立即更新列表来获得最新的自动代理规则列表。

  15. 确认 SwitchySharp 中设置为自动切换模式，确认 goagent 客户端已运行，输入墙外热门的地址，去看看外边的世界吧！


注意事项

  • 如果在一台新的笔记本上配置，发现在访问 https 时需要导入 goagent 的证书才可以，如果是 Vista、Win7 之类的系统，默认没有管理员权限无法导入将导致这些站点不可访问。
    特附上解决方法，先关闭 Chrome，然后在图标上右键重新以管理员身份打开。

  • 在 chrome 的高级选项中，先点击管理证书，弹出窗口中选择导入，找到 goagent/local/CA.crt，导入时系统自动选择证书存储位置即可。这时，就可以正常的访问 https 的加密页面了，比如 Google Plus。
